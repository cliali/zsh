## 1. install zsh

### 1.1 fedora

```bash
sudo dnf install zsh
```

### 1.2 ubuntu

```bash
sudo apt install zsh
```

### 1.3 cent os

```bash
sudo yum install zsh
```

### 1.4 arch

```bash
sudo pacman -S zsh
```


## 2. give permission to execute the script

```bash
sudo chmod +x zsh.sh
```


## Notice

full screen the terminal for powerlevel10k config


## 3. run the script

```bash
./zsh.sh
```
