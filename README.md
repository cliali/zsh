## 1. install zsh

### 1.1 fedora

```bash
sudo dnf install zsh
```

### 1.2 ubuntu

```bash
sudo apt install zsh
```

### 1.3 cent os

```bash
sudo yum install zsh
```

### 1.4 arch

```bash
sudo pacman -S zsh
```


## Notice

full screen the terminal for powerlevel10k config


## 2. Run the script

```bash
wget -q https://raw.githubusercontent.com/cliali/zsh/main/zsh.sh -O zsh.sh 

chmod +x zsh.sh

./zsh.sh
```
